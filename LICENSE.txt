This extension, "Ethereal Nexus", is primarily developed by Mohamed Metwally. 
It is a collection of custom VS Code themes with unique color palettes and design.

Some code for editor line color changes is adapted from "Dark One Pro" theme. 
This use is limited and only applies to certain themes within this collection.

All other code and theme definitions are original or derived from VS Code's default themes, which are open source.
